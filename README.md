# Functions - Part 1

## Learned

- A function is a reusable block of code.
- Functions are created using `def`.
- A function executes only when it is called.
- Python reads the program from top to bottom.
- After a function finishes, execution returns to the next line after the function call.

## New Keywords

- def
- Function Definition
- Function Call

## Mentor Notes

- Defining a function does not execute it.
- Always give meaningful function names.
